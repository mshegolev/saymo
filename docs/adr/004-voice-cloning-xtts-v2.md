# ADR-004: Voice Cloning with Coqui TTS XTTS v2

**Status:** Accepted
**Date:** 2026-04-16
**Author:** Mikhail Shchegolev

## Context

Стендап-отчёт должен звучать голосом пользователя, а не синтетическим TTS. Рассматривались:

| Решение | Русский | Цена | Качество | Латентность | Локальное |
|---------|---------|------|----------|-------------|-----------|
| ElevenLabs | Да | $22+/мес | 99% | 75ms | Нет |
| Coqui XTTS v2 | Да | Бесплатно | ~85-90% | ~30с/текст | Да |
| Bark (Suno) | Частично | Бесплатно | ~80% | Медленно | Да |
| Piper TTS | Да | Бесплатно | ~80% (no clone) | <1с | Да |
| macOS say | Да | Бесплатно | Синтетический | <1с | Да |

## Decision

**Coqui TTS XTTS v2** как основной engine + **Piper** и **macOS say** как fallback.

### Voice Sample
- Записан 5-минутный семпл (`~/.saymo/voice_samples/voice_sample.wav`)
- Содержит реальную рабочую лексику: пайплайн, стейдж, деплой, хотфикс, NetSuite, Spark, Hive
- Записан через Plantronics гарнитуру (тот же микрофон что на реальных звонках)

### Текстовый препроцессинг (text_normalizer.py)
Перед TTS текст нормализуется:
- Аббревиатуры → русская фонетика: `NS2` → `эн-эс-два`, `QA` → `кью-эй`
- IT-термины → произношение: `smoke` → `смоук`, `stage` → `стейдж`
- Удаление build-версий: `v.2604101636` → (пусто)
- Удаление JIRA ID: `DATA-15989` → (пусто)
- Числа → русские слова: `15` → `пятнадцать`

### TTS Engine Hierarchy
```
coqui_clone → piper → macos_say (fallback chain)
```

## Consequences

**Positive:**
- Голос узнаваем как пользовательский (~85-90% похожесть с 5-мин семплом)
- Полностью бесплатно и локально
- Text normalizer обеспечивает корректное произношение IT-терминов

**Negative:**
- Генерация медленная (~30-90с) → решено через pre-generation (`prepare`)
- Требует arm64 терминал (Rosetta x86_64 не поддерживает PyTorch 2.6+)
- Модель ~2GB на диске
- Качество ниже ElevenLabs — можно улучшить через RVC (future)

## Future Improvements

- **RVC (Retrieval-based Voice Conversion)**: тренировка конвертера голоса на 10-20 мин аудио → более точный клон
- **Fine-tune XTTS v2**: дообучение на 2-6 часах аудио → максимальное качество
- **Sentence-level review** (`saymo review`): уже реализовано, позволяет перегенерировать плохие фрагменты
