# Plan 11-02: Backend-Neutral Diarization Result Contracts

## Goal

Create Saymo-owned dataclasses and JSON helpers for diarization segments and
results so later phases can store sidecars without backend-specific objects.

## Requirements

- DIAR-01
- DIAR-03

## Tasks

- Add immutable dataclasses for diarization segments, results, and
  availability diagnostics.
- Add validation/normalization for speaker ids, times, confidence, engine, and
  model metadata.
- Add JSON serialization/deserialization helpers.
- Ensure optional backend imports are isolated behind functions.
