# Plan 06-02: Train Lightweight Local Trigger Classifier

## Requirements

- CLF-02: User can train a lightweight local trigger classifier from
  accepted/rejected samples after seeing a data-sufficiency check.

## Tasks

- Add a dependency-free classifier module for transcript and sample metadata
  features.
- Refuse training when total labeled samples or per-class counts are below
  configured thresholds.
- Store profile-specific classifier artifacts as local JSON under
  `~/.saymo/models/trigger_classifier/`.
- Add `trigger-classifier train|inspect|delete` CLI commands.

## Verification

- Unit tests for threshold refusal, prediction, save, and load behavior.
- CLI tests for train, inspect, and delete.
