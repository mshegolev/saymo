# Plan 09-02: Add Category Relabel Command

## Requirements

- REV-02: User can correct a sample category from the CLI, including
  `asked_to_speak`, `mentioned_me`, `question`, `speech`, and `silence`,
  without manually editing JSON.

## Tasks

- Add a safe metadata writer for category changes.
- Move adjacent WAV and JSON files into the target category folder.
- Preserve session metadata, speaker labels, and answer decisions.
- Refuse missing adjacent WAV only when it would make the sample undiscoverable;
  otherwise keep JSON correction possible for legacy metadata.

## Verification

- CLI tests for moving JSON+WAV and updating category metadata.
