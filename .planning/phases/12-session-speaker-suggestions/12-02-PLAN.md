# Plan 12-02: Store Diarization Sidecars And Sample Suggestions

## Goal

Write session diarization sidecars containing raw segments and per-sample
speaker suggestions without mutating sample metadata.

## Requirements

- DIAR-04
- SPKR-01

## Tasks

- Add sidecar path/load/write helpers.
- Match diarization segments to sample sequence windows by overlap.
- Store suggestions with current manual label, inferred speaker id, mapped
  label if present, confidence, overlap, and review status.
- Keep sessions with no sidecar compatible with existing list/review/report
  flows.
